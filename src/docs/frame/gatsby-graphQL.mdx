## GraphQL

<WordTitle type='define'>GraphQL</WordTitle>

GraphQL 既是一种用于 API 的查询语言也是一个满足你数据查询的运行时。

**特点**

**➙ 请求的数据刚刚好**

只获取请求的字段。

**➙ 获取多个资源，只用一个请求**

**➙ GraphQL是描述型语言，可以自由描述所有可能的类型系统，便于维护**

<br/>

<QuestionTitle>GraphQL 与 Restful API 的区别</QuestionTitle>

Restful API 针对的是通用系统框架，不受语言类型的限制，本质是用定义的URL，通过API接口获取资源。

|                       | GraphQL                | Restful API                          |
| ----------------------| ---------------------- | ------------------------------------ |
| 返回资源               | 一次获取多个             | 一个接口一个资源                        |
| 资源区分               | 用类型区分资源           |  用 URL 区分资源                     |
