## 规范化标题 RegularTitle

本博客将一些常用的标题类型封装成组件，其中可选种类为：

- **阐述 show**  - 对关键概念含义以及关系的解释、解释原因推导过程。
- **实例 example** - 为说明概念而准备的实例。
- **注意点 note** - 对关键点的提示。
- **总结 summary** - 对知识整体的总结。

| Key                    | Required | Description                |
| ---------------------- | -------- | -------------------------- |
| type                   | Yes      | tag type                   |
| title                  | No       | tag title                  |

<RegularTitle type='example' title='Hello World'/>

```javascript
    console.log('Hello World')
```
